Email Setup In Local Machine

A fresh local machine needs some configuration for handle emails.

  • Step One

    Open “Run” in windows and type “optionalfeatures” then hit enter.

    Turn on "Telnet Client” Feature.

  • Step Two

    Update PHP's CA Certificate Path.

    • Go to https://curl.se/ca/cacert.pem and download cacert.pem.
    • Save it to a known location, e.g., C:\xampp\php\extras\ssl\cacert.pem.

  • Step Three
    • Open php.ini and search “;extension=php_imap.dll” and uncomment it.
    • Search “curl.cainfo” and replace it with
    curl.cainfo = "C:\xampp\php\extras\ssl\cacert.pem"

    • Search “openssl.cafile” and replace it with 
    openssl.cafile = "C:\xampp\php\extras\ssl\cacert.pem"
    Restart Apache & Test Again

Email Synchronization

Our CRM email system stores emails in its database. As a result, when emails are deleted from external email managers such as Gmail, Outlook, or IONOS, they are not automatically removed from the CRM.

To address this, we have developed a PHP script and scheduled it as a CRON job for automated execution. This script runs continuously, comparing emails in the CRM with those on the email server and deleting any that no longer exist on the server.

  • For a detailed understanding of its functionality, please refer to the cron_delete_email.php file.
    
set_time_limit(300); // 5-minute timeout
include_once('inc/config.php');

// Use a single query to fetch all necessary data with JOINs
$currentDate = date("Y-m-d");
$query = "
    SELECT 
        ed.id AS account_id, ed.username, ed.password,
        ef.id AS folder_id, ef.name, ef.inbox_url,
        ec.email_uuid
    FROM email_dashboards ed
    LEFT JOIN email_folders ef ON ef.email_id = ed.id
    LEFT JOIN email_content ec ON ec.folder_id = ef.id AND DATE(ec.date) = '$currentDate'
    WHERE ec.email_uuid IS NOT NULL
";
$result = mysql_query($query) or die("Main query failed: " . mysql_error());

// Group data in memory for processing
$accounts = [];
while ($row = mysql_fetch_object($result)) {
    if (!is_numeric($row->account_id) || !is_numeric($row->folder_id)) {
        echo "Invalid ID - Account: " . htmlspecialchars($row->account_id) . 
             ", Folder: " . htmlspecialchars($row->folder_id) . '<br>';
        continue;
    }

    $account_id = (int)$row->account_id;
    $folder_id = (int)$row->folder_id;

    // Structure: [account_id][folder_id] => [details, email_uuids]
    if (!isset($accounts[$account_id])) {
        $accounts[$account_id] = [
            'username' => $row->username,
            'password' => $row->password,
            'folders' => []
        ];
    }
    if (!isset($accounts[$account_id]['folders'][$folder_id])) {
        $accounts[$account_id]['folders'][$folder_id] = [
            'name' => $row->name,
            'inbox_url' => $row->inbox_url,
            'email_uuids' => []
        ];
    }
    $accounts[$account_id]['folders'][$folder_id]['email_uuids'][] = (int)$row->email_uuid;
}

// mysql_free_result($result);

// Process each account and folder
foreach ($accounts as $account_id => $account) {
    foreach ($account['folders'] as $folder_id => $folder) {
        if (empty($folder['email_uuids'])) {
            continue; // Skip if no emails to process
        }

        echo $account['username'] . '<br>';
        echo $folder['name'] . '<br>';

        // Open IMAP connection
        $iMap = imap_open($folder['inbox_url'], $account['username'], $account['password']);
        if (!$iMap) {
            echo "IMAP not connected for folder '{$folder['inbox_url']}': " . imap_last_error() . "<br>";
            continue;
        }

        // Fetch existing emails in one go
        $uidString = implode(',', $folder['email_uuids']);
        $existingEmails = imap_fetch_overview($iMap, $uidString, FT_UID);
        $existingUIDs = $existingEmails ? array_map(function($email) { return $email->uid; }, $existingEmails) : [];

        // Find UIDs to delete (not in IMAP)
        $uidsToDelete = array_diff($folder['email_uuids'], $existingUIDs);

        if (!empty($uidsToDelete)) {
            // Batch delete with a single query
            $uidStringForDelete = implode(',', array_map('intval', $uidsToDelete));
            $deleteQuery = "DELETE FROM email_content WHERE email_uuid IN ($uidStringForDelete)";
            mysql_query($deleteQuery) or die("Batch delete failed: " . mysql_error());
        }

        imap_close($iMap);
    }
}

echo 'Done';
